# 本番アップ手順（普通のブラウザ・スマホから開けるようにする）

このダッシュボードは **HTML と JavaScript だけの静的サイト**なので、無料のホスティングにファイルを置くだけで、
PC・スマホどちらの普通のブラウザからでも URL で開けるようになります（アプリのインストール不要）。

---

## 事前準備（1回だけ）：全員が自動でスプレッドシートに繋がるようにする

そのままアップすると、開いた人ごとに「接続設定」で GAS の URL を入れる必要が出てしまいます。
それを避けるため、**アップ前に GAS の URL をコードに1か所貼り付けます。**

1. `app.js` をテキストエディタで開く（右クリック →「このアプリケーションで開く」→「テキストエディット」）
2. 先頭のほうにある次の行を探す：
   ```js
   const DEFAULT_API_URL = '';
   ```
3. `''` の中に、あなたの GAS ウェブアプリURL（…/exec）を貼り付けて保存：
   ```js
   const DEFAULT_API_URL = 'https://script.google.com/macros/s/XXXXXXXX/exec';
   ```

これで、URL を開いた全員（PC・スマホ）が最初からスプレッドシートに接続され、
ログイン画面が出るようになります（各自の接続設定は不要）。

> ※ここを空 `''` のままにすると、従来どおり各ブラウザで「接続設定（社長のみ）」から設定する方式になります。

---

## アップする方法（いちばん簡単：Netlify Drop）

**アカウント登録なしでも試せて、HTTPS の URL がすぐ発行されます。スマホでもそのまま開けます。**

1. PC で https://app.netlify.com/drop を開く
2. `tori-dashboard` フォルダを、そのページに**ドラッグ＆ドロップ**する
   - アップするのに必要なのは **`index.html` / `app.js` / `sample-data.js`** の3つ
   - `gas/` フォルダ・`README.md`・`*.bak`・`*.md` は無くてもOK（あっても害はありません）
3. 数十秒で `https://ランダムな名前.netlify.app` のような URL が発行されます
4. その URL を PC・スマホのブラウザで開けば完成。スマホはホーム画面に追加すればアプリのように使えます

> アカウントを作って「サイト名」を決めると、`https://tori-dashboard.netlify.app` のような固定URLにできます。
> 更新したいときは、同じ画面に新しいフォルダをドロップし直すだけです。

### Netlifyの無料枠が切れた／使えないとき（他の無料ホスティング）

※ Netlify Drop はアカウント無しだと**一時的なサイト**で、しばらくすると消えます。以下は**無料で永続**します。

**おすすめ①：Cloudflare Pages（無料・永続・ドラッグ&ドロップ可）**
1. https://dash.cloudflare.com でアカウント作成（無料・クレカ不要）
2. 左メニュー **Workers & Pages → Create → Pages → Upload assets（直接アップロード）**
3. `tori-dashboard` フォルダ（またはindex.html/app.js/sample-data.jsの3つ）をアップロード
4. `https://〇〇.pages.dev` のHTTPS URLが発行 → PC・スマホで開ける

**おすすめ②：GitHub Pages（無料・永続）**
1. GitHubアカウントを作る
2. 新しいリポジトリを作成 → 3つのファイルをアップロード
3. Settings → Pages → Branch を main に設定 → `https://ユーザー名.github.io/リポジトリ名/`

**その他：Firebase Hosting（Google・無料）／Vercel（無料）** も同様に静的サイトを公開できます。

> どれも「静的サイトを置くだけ」で、月額もドメイン購入も不要です。
> Cloudflare Pages が画面のドラッグ&ドロップで完結して分かりやすいので、まずはそれがおすすめです。
> 手順で迷ったら、どの方法にするか教えてください。その方法だけを画面つきで細かく案内します。

## アップ後のチェック

- [ ] スマホのブラウザで URL を開ける
- [ ] ログイン画面が出る（＝ GAS URL が効いている）。デモ用の「サンプルデータモード」表示になっていないか確認
- [ ] 各権限のID/パスワードでログインできる（スプレッドシートの「アカウント」シートで管理）
- [ ] 12店舗の実データが出る／数字が正しい
- [ ] スマホで横スクロールせず見られる（表は指で横スワイプでスクロール）

---

## セキュリティの注意（社内利用前提）

- URL は「知っている人だけがアクセス」する前提です。**ログインしないと中身は見えません**（トークン認証・12時間で失効）。
- GAS の URL とスプレッドシートの共有範囲は、**社外に出さない**でください。
- 初期パスワード（shacho/tori2026 など）は必ず変更してください（アカウント管理タブ、または「アカウント」シート）。
- ダミーの検索エンジン登録などは不要ですが、URL を推測されにくい名前にしておくと安心です。
